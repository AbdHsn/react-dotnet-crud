# Run command

- 1. Run DotNet Core: dotnet run
- 2. Run React UI:
  - 2.1 npm i -f //install node dependancies forcefully
  - 2.2 npm start // run react ui 

## Environment Information

- **Node:** 18.10.0
- **Package Manager:** npm 8.19.2
- **OS:** Any
